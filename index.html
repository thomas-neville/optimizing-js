Optimizing Javascript and Page Load Times

This essay will discuss high level approaches to optimizing Javascript for more effient ways of executing code and reducing page load times. It will not go into detail about specific libraries and frameworks for Javascript such as jQuery or AngularJS. This essay will also assume that Javascript is needed for the web components being discussed.
